## Стек технологий

### Backend (HTTP-сервис)
- Node.js + TypeScript
- Fastify — REST API
- @fastify/mongodb — интеграция с MongoDB
- Redis + Bull — кэш и очередь задач (отложенные публикации)
- Axios — HTTP-клиент для интеграции с внешними ИИ-сервисами
- dotenv — переменные окружения

### Парсер Telegram
- Node.js + TypeScript
- Telegram MTProto / Bot API (библиотеки в parser/src)
- Подключение к MongoDB и Redis

### ИИ-интеграция
- Timeweb Cloud AI (HTTP API) — уникализация/рерайт текста
- Потенциально Yandex GPT / другие LLM API

### Frontend (админ-панель)
- Vue.js 3 + TypeScript
- Vue Router — маршрутизация
- Vite — сборка и dev-сервер
- Компонентный подход (views + components)

### БД и инфраструктура
- MongoDB — основная БД (посты, каналы, категории, пользователи, логи, аналитика)
- Mongo Express — веб-интерфейс для MongoDB
- Redis — кэш/очередь задач
- Docker / Docker Compose — контейнеризация всех сервисов
- Git — контроль версий

## Требования к оформлению (.docx по ГОСТу)

### Общие параметры
- Формат А4
- Поля: сверху 20мм, снизу 20мм, справа 15мм, слева 30мм
- Шрифт: Times New Roman, 14 пт
- Межстрочный интервал: 1.5
- Абзацный отступ: 1.25 см
- Выравнивание: по ширине
- Не добавлять интервал между абзацами одного стиля
- Цвет шрифта: чёрный
- Печать на одной стороне

### Нумерация страниц
- Арабские цифры, внизу по центру
- Нумерация начинается с титульного листа
- На титульном листе, задании и аннотации номера НЕ ставятся
- В оглавление титульный лист, задание, аннотация и само оглавление НЕ выносятся

### Заголовки
- Заголовки разделов: по центру, без абзацного отступа, без точки в конце, ПРОПИСНЫЕ БУКВЫ, полужирный
- Заголовки подразделов: с прописной буквы, без точки в конце, полужирный
- После номера раздела/подраздела перед заголовком точка НЕ ставится
- Переносы в заголовках запрещены
- Разделы начинаются с новой страницы
- Подразделы идут друг за другом с дополнительным интервалом

### Не нумеруются
- Аннотация, Введение, Заключение, Список источников, Список сокращений

### Списки
- Первого порядка (1. 2. 3.): прописная буква в начале, точка в конце
- Второго порядка (а) б) •): строчная буква в начале, точка с запятой в конце, точка в конце всего списка
- Вложенные: строчная буква, запятая в конце, точка с запятой в конце вложенного списка

### Рисунки и таблицы
- Рисунки: подпись снизу по центру, без абзацного отступа. Формат: «Рисунок N – Название»
- Таблицы: название сверху слева, без абзацного отступа. Формат: «Таблица N – Название»
- Точка в конце подписи НЕ ставится
- На все таблицы и рисунки должны быть ссылки в тексте
- Располагаются после абзаца, в котором упоминаются впервые
- Ссылки должны быть кликабельными

### Выделения
- Полужирный — только для заголовков
- Курсив — для терминов (кириллица и латиница)

### Приложения
- Нумеруются кириллическими буквами: Приложение А, Приложение Б
- Рисунки/таблицы в приложениях: «Рисунок А.1 – Название»
